# Introduce

## 名词解释

* **机器学习**
  * **实际上就是寻找X 到 Y之间的一种对应关系 F，这种对应关系F 称之为一个模型；然而机器学习就是对很多个X 和 Y，通过对模型的训练，然后去发现这个最优的模型的过程。**
* **拟合**
  * **过拟合**
    * **是指训练好的模型F，对某些特征要求过于苛刻**
  * **欠拟合**
    * **与过拟合相反**
  * **Just Fit**
    * **刚刚好**
* **特征工程**
  * **一维特征：只有一个自变量的关系式**

                       **e.g. f\(衣服厚度\) = 温度      只有一个温度自变量**

  * **二维特征：**

                       **e.g. f\(衣服厚度\) = 温度 + 风速      温度和风速两个自变量**
* **划分训练集和测试集**
* **评价模型**
